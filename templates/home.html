Flask Framework

Flask é um micro framework web minimalista escrito em Python. Projetado para ser simples, mas extensível, Flask fornece o essencial para desenvolvimento web sem impor dependências desnecessárias.

Comece a Codar Docs

Por que Flask?

Minimalista

Núcleo pequeno e simples - apenas o essencial para aplicações web. Sem sobrecarga desnecessária ou decisões arquiteturais impostas.

Extensível

Arquitetura modular permite adicionar apenas as funcionalidades que você precisa através de extensões bem integradas.

Produtivo

Configuração mínima e curva de aprendizado suave permitem começar a desenvolver rapidamente.

Pythonico

API simples e intuitiva que segue as melhores práticas e padrões Python.

Seguro

Proteção contra CSRF, saneamento de dados e boas práticas de segurança implementadas.

WSGI

Compatível com WSGI, permitindo deploy em diversos servidores como Gunicorn, uWSGI ou Waitress.

Primeiros Passos

Instalação

Configure seu ambiente e instale o Flask em poucos passos:

# Crie e ative um ambiente virtual (recomendado)
python -m venv venv
source venv/bin/activate # Linux/Mac
# ou .\venv\Scripts\activate no Windows

# Instale o Flask
pip install flask

Estrutura Básica

Um aplicativo Flask mínimo tem esta estrutura:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def home():
    return "Olá, Flask!"

if __name__ == '__main__':
    app.run(debug=True)

Executando Seu App

Salve o código em app.py e execute:

python app.py

Acesse http://127.0.0.1:5000 no navegador para ver seu aplicativo em ação.

Dicas Rápidas
Modo Desenvolvimento

Ative o modo debug para:

  • Recarregamento automático ao editar código
  • Páginas de erro detalhadas
  • Debugger interativo para troubleshooting
# Ative via código
app.run(debug=True)

# Ou via variável de ambiente
export FLASK_DEBUG=1 # Linux/Mac
set FLASK_DEBUG=1 # Windows

Exemplos Práticos

Roteamento Dinâmico

@app.route('/user/<username>')
def show_user(username):
    return f'Usuário: {username}'

@app.route('/post/<int:post_id>')
def show_post(post_id):
    return f'Post #{post_id}'

@app.route('/path/<path:subpath>')
def show_subpath(subpath):
    return f'Subpath: {subpath}'

Templates Jinja2

# app.py
from flask import render_template

@app.route('/hello/<name>')
def hello(name):
    return render_template('hello.html', name=name)

# templates/hello.html
<!DOCTYPE html>
<html>
<head>
    <title>Hello Flask</title>
</head>
<body>
    <h1>Olá, {{ name }}!</h1>
    {% if name == 'admin' %}
        <p>Acesso especial concedido</p>
    {% endif %}
</body>
</html>

Manipulação de Requisições

from flask import request

@app.route('/login', methods=['GET', 'POST'])
def login():
    if request.method == 'POST':
        return f"Usuário {request.form['username']} logado!"
    return '''
        <form method="post">
            Usuário: <input type="text" name="username">
            <button type="submit">Login</button>
        </form>
    '''

API REST Simples

from flask import Flask, jsonify, request

app = Flask(__name__)

tasks = [
    {'id': 1, 'title': 'Aprender Flask', 'done': False},
    {'id': 2, 'title': 'Construir API', 'done': False}
]

@app.route('/tasks', methods=['GET'])
def get_tasks():
    return jsonify({'tasks': tasks})

@app.route('/tasks', methods=['POST'])
def create_task():
    if not request.json or not 'title' in request.json:
        return jsonify({'error': 'Bad request'}), 400
    task = {
        'id': tasks[-1]['id'] + 1,
        'title': request.json['title'],
        'done': False
    }
    tasks.append(task)
    return jsonify({'task': task}), 201
